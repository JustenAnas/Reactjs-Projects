### Project 5 - Image Enhancer

An AI-powered image enhancement application that improves image quality and resolution through a simple and user-friendly interface.

#### Features

* Upload images for enhancement
* AI-powered image processing
* Image quality improvement
* Enhanced image preview
* Download enhanced images
* Fast and responsive user experience
* Modern and clean UI

#### Tech Stack

* React
* Node.js
* Express.js
* JavaScript
* AI Image Enhancement API
* CSS
