 ### Project 2 - GTA VI Landing Page
A visually engaging GTA VI-inspired landing page built with React and GSAP, focused on modern animations and immersive user experience.

#### Features:

- Smooth scroll-based animations
- GSAP-powered transitions and effects
- Interactive hero section
- Animated text and image reveals
- Responsive layout
- Modern gaming-inspired UI
- Optimized performance and smooth interactions

#### Tech Stack:

- React
- GSAP
- JavaScript
- CSS
